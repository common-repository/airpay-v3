Seamlessly integrate Airpay's payment gateway for secure online transactions on your WordPress site.

== Description ==
Say goodbye to the complexity of payment procedures hindering your online business. Introducing airpay, your go-to fintech solution for transforming your WooCommerce store's payment experience.
 
Airpay offers a unique combination of features, including a responsive payments page, robust failover mechanisms, multi-platform integration, customizable payment experiences, and comprehensive multi-currency support.
These aspects collectively contribute to providing merchants with a distinct advantage in terms of user experience, reliability, and global expansion opportunities.

==Benefits==
- Responsive payments page
- Auto failover & dynamic routing for payment transactions
- Multi-platform integration
- Native payment app experience or PG redirection
- Multi-currency support

==Requirements==
* PHP 5.1 or later
* wordpress 4.7 to 6.4.3
* WooCommerce 2.5 to 6.0.0 plugin
* Merchant Account of airpay payment services.
* An Editor to Edit Code.
* a web-browser

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'Settings' or 'Tools' menu to configure the plugin settings (if applicable).
